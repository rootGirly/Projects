## 🛡️ Security Skills learned

### Infrastructure Security
- **System Hardening:** CIS Benchmarks, UFW firewall, Fail2Ban
- **Network Security:** DNS sinkholing, port restriction, VLAN isolation concepts
- **Virtualization:** KVM/LXC isolation, secure VM configuration, snapshot management

### Application Security
- **Web Security:** Security headers (CSP, HSTS, X-Frame-Options), HTTPS enforcement
- **Authentication:** Secret management, environment variable security, API key handling
- **Data Protection:** Encrypted backups, database security, TLS everywhere

### Automation & DevSecOps
- **Infrastructure as Code:** Ansible playbooks, Docker Compose
- **CI/CD Concepts:** Automated deployments, configuration management
- **Monitoring:** Log analysis, audit trails, alerting concepts

### Privacy & Compliance
- **GDPR-Friendly:** Self-hosted alternatives to third-party services
- **Data Sovereignty:** Full control over data location and retention
- **Open Source:** Preference for auditable, community-driven solutions
